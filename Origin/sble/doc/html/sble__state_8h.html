Detailed Description

The state of SBLE.

The state of BLE consists of a global variable of type sble_driver_state, called dstate. It contains properties that are global to all connections, and at its cons[]-array, properties that are individual to each acitve connection. This File is highly plattform dependant and has to be adapted if SBLE is to be ported to a new plattform.
