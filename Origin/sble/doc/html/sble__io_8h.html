Detailed Description

Input/Output via Serial UART to BLE112-Device.

This file does the I/O to the BLE112 or BLED112-Device. The header-file is platform-independant, whereas the corresponding .c-files is highly platform-dependant. It must be reimplemented for any platform. Naming Convention for implementations of this header: sble_io_[platform].c
