Detailed Description

Purely virtual (abstract) class that needs to be implemented to get sensor data via the HE2mT SDI The class "genericParamInterface" is for debugging purposes, only. It is not part of the interface. The standard C++ installation comes with a dummy implementation of genericParamInterface as the accsensor_server - application expects it to be there. In Java on a mobile phone, genericParamInterface needs not to be implemented at all.
